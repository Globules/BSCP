## Labs tips

Every lab from authentication has:

- login functionality (`/login`)
- credential-based access (username / password)
- session management (cookies, tokens)
- brute-force or enumeration possibilities
- flawed authentication logic or missing protections
- interaction with account-related features (reset password, 2FA, etc.)
