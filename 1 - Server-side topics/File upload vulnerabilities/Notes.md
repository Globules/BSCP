## Labs tips

Every lab from file upload vulnerabilities has:

- upload functionality (profile picture, avatar, document, etc.)
- server-side validation (MIME type, extension, content)
- accessible upload directory (e.g. `/files/`, `/images/`)
- possibility to execute uploaded files (web shell)
